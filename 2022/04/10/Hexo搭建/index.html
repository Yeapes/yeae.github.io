Hexo搭建
更新于 2022-04-11 热度 1 喜欢 0

关键词

  • Node 官网链接
  • Git 官网链接
  • Hexo
  • Github

1.安装Node Git并验证

  • 验证：在命令行下分别执行 node-v npm -v
    git-v
  • 出现对应版本号即为成功

2.更换淘宝镜像源

  • 命令：npm install -g cnpm –registry==https://registry.npm.taobao.org
  • 验证：cnpm -v

3.安装hexo

  • 命令：cnpm install -g hexo-cli
  • 验证：hexo -v

4.创建Github仓库

  • user.github.io
  • 配置SSH
    1.在C:\Users\用户名 文件夹下右键打开Git Bash Here输入命令：ssh-keygen -t rsa -C 邮件地址
    2.三次回车
    3. .ssh文件内复制公钥内容添加至Github SSH and GPG keys
    4.验证: ssh -T git@github.com （不用改动）
    5. git config –global user.name “用户名”
    git config –global user.email “注册邮箱”

5.使用 hexo 搭建博客

  • 创建文件夹放置博客内容，此文件夹内右键 Git Bash Here
  • hexo初始化：hexo init
  • 预览: hexo g && hexo s

6.将hexo部署到GitHub

  • 博客目录下安装插件 :npm install hexo-deployer-git –save
  • 编辑博客根目录下 _config.yml 文件, 在文件末尾添加如下内容：
    deploy:
    type: git
    repo: https://github.com/user/user.github.io.git
    branch: master
    （user改为个人昵称；冒号之后存在空格）
  • hexo d
  • 网站访问 user.github.io
